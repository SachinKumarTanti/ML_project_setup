About the Project

This machine learning project aims to predict student math scores based on various input features such as gender, race/ethnicity, parental level of education, lunch type, test preparation course completion, and reading/writing scores. The model is trained on a dataset of student performance and uses preprocessing techniques like one-hot encoding and standard scaling to handle categorical and numerical data. Built with Python, Flask, scikit-learn, and CatBoost, this application provides an interactive way to explore predictive analytics in education.
